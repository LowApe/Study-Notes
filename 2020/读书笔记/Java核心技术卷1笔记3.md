### 多态

> 在Java程序设计中，任何变量对象是多态的，变量既可以引用一个本身的类，也可以引用该类的任意一个子类对象。

```java
/* 子类数组的引用可以转换超类数组的引用，而不需要强制类型转换*/
Manager[] managers = new Manager[10];
Employee[] emploes = managers;
emploes[0] = new Employee();
managers[0].getManager(); // 因为引用同一个数组，所以指向同一个数组，因为getManager方法只有Manager
Exception in thread "main" java.lang.ArrayStoreException:
```

> ⚠️注意：一定要记住数组的所处的类型，如发生上述情况(emploes 和 managers 引用同一个数组)会出现异常
