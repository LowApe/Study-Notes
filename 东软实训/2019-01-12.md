# 行级元素和块级元素

> 场景:有些标签默认行着排布,有些标签纵向排布

```css
*{
    display:block;
    //默认 布局 块级元素
    display: inline;
    //行级元素
    display: inline-block;
    //两者兼顾,高宽生效
}

```
添加下划线
```css
{
    border-bottom:1px solid red;
}
```
