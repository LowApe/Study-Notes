# 使用脚手架工具创建 React 项目
脚手架:
1. Create React App [facebook官方-入门]
2. Codesandbox
3. Rekit

为什么使用脚手架?
> 前端项目通常引入多个技术站是很复杂的
1. React UI开发
2. Redux 状态的管理
3. React/Router 路由管理
4. BABEL 新的 JavaScript 特性翻译游览器旧的代码
5. webpack 打包
6. ESLint 语法检查
