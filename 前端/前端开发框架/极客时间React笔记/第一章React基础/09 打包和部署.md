# 打包和部署
> 为什么需要打包?
1. 编译 ES6 语法特性,编译 JSX
2. 整合资源,如图片,Less/Sass
3. 优化代码的体积

![](http://ww1.sinaimg.cn/large/006rAlqhly1g0ncja42v7j30g508lwgn.jpg)
> 插件方式插入 webpack 需要单独学习，配置等

> 打包注意事项
1. 设置 nodejs 环境为 production(开发模式有提示)
2. 禁用开发时专用代码,比如 logger(提示代码)
3. 设置应用根路径(加载资源)
