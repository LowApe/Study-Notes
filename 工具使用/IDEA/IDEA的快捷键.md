# IDEA 的快捷键

> 本文记录开发过程中可以提高开发效率的快捷键

| 快捷键效果   | MAC快捷键组合          | WIN快捷键组合 |
| ------------ | ---------------------- | ------------- |
| 多选输入位置 | `option`+`shift`+鼠标  |               |
| 查找         | `Command`+`F`          |               |
| 查找替换     | `Command`+`R`          |               |
| 代码移动     | `Command`+`shift`+箭头 |               |
