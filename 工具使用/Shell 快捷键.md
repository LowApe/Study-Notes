# Shell 快捷键

- `ctrl+a`快速移动到开头
- `ctrl+e`快速移动到结尾
- `ctrl+u//`删除光标前面的全部内容
- `ctrl+k`删除光标后面的全部内容
- `ctrl+w`删除光标前面的单词(空格)
- `ctrl+h`删除光标前面单个字符
- `ctrl+l`清理屏幕，有点类似于 `clear`
- `ctrl+ p` 向上
- `ctrl+ n` 向下
- `ctrl+b`向左
- `ctrl+o/j`类似于 回车
